# Planoptico

### Programa OCR para arq

Programa que analiza y exporta medidas de planos en pdf

# Realese 1
* OCR library para leer planos en pdf
* suma metros lineales de paredes
* computa areas y perimetros
* cotas y polilineas
* resultado en csv

# Realese 2
* machine learning
* base de datos
* gui para devs
